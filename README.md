# MapLibre GeoJSON Sample

A simple map application using MapLibre GL JS with GeoJSON data.  
Displays multiple points on an OpenStreetMap base layer.

## Files

- `index.html` - Map viewer using MapLibre GL JS
- `data.js` - GeoJSON data (Tokyo Station, Shibuya, Shinjuku)

## Usage

Clone the repo and open `index.html` in your browser.  
(No build tools required)

## Tech

- [MapLibre GL JS](https://maplibre.org/) v3.6.2
- OpenStreetMap

# MapLibre GeoJSON サンプル

MapLibre GL JS と GeoJSON を使ったシンプルな地図アプリです。  
OpenStreetMap をベースに、複数のポイントを表示します。

## ファイル構成

- `index.html` - MapLibre GL JS を使った地図表示
- `data.js` - GeoJSON データ（東京駅・渋谷・新宿）

## 使い方

リポジトリをクローンして `index.html` をブラウザで開くだけ。  
ビルド不要です。

## 使用技術

- [MapLibre GL JS](https://maplibre.org/) v3.6.2
- OpenStreetMap
